# One-click Script Installation

1. Login to your VPS by SSH, then run this followed command:

```shell
bash <(wget -qO- https://raw.githubusercontent.com/sing-web/x-ui/main/install.sh)
```

2. For security reasons, you must set panel port, username and password. Type "y" for set them or type "n" for default settings (Username and password are admin, panel port it's 54321)
3. If everything it's ok, you will get IPv4 / IPv6 access address shown on your script's result
